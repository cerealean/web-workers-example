Main Thread vs. Worker Thread

Run In Main Thread

The calculations will be ran in the main thread, blocking the UI.



Run In Main Thread With Async

The calculations will be ran in the main thread, blocking it. You'll see that, even though we are running async, the UI is still blocked.



Run In Worker Thread

The calculations will be ran in the worker thread, letting the main thread concentrate on processing the UI.
